# Templates Folder

This folder contains Obsidian Templater templates for the DPD-001 project.

## Quick Access Templates

- **`new_vhdl_component.md`** ‚≠ê - Create paired VHDL component files (recommended)
- **`base.md`** - Standard base template with frontmatter
- **`00_times_startup_hook.md`** - Startup hook (auto-runs on Obsidian startup)

## Manual Templates

- **`vhdl_file.vhd`** - Template for `.vhd` files
- **`vhdl_doc.vhd.md`** - Template for `.vhd.md` files

## Documentation

See `../docs/` folder for:
- VHDL pair template documentation
- Templater configuration guide
- Frontmatter time fields documentation

## User Scripts

The `user_scripts/` folder contains JavaScript modules used by templates. These are **not templates** and should not appear in the template selection menu.

**Note:** Configure Templater's "User scripts folder location" to point to this `user_scripts/` folder.
